Donjon des Ombres

Type : Coopération

Niveau recommandé : 20-30

Difficulté : ★★☆☆☆ (Modérée)

Récompenses : Équipement rare, ressources magiques

Description : Un ancien temple caché dans les montagnes, rempli de créatures obscures et de trésors oubliés. Les joueurs doivent travailler en équipe pour vaincre les boss et résoudre des énigmes.

Conseil : Privilégiez les classes à distance pour éviter les pièges et les embuscades dans les couloirs sombres.

Forteresse du Dragon

Type : Groupe

Niveau recommandé : 30-40

Difficulté : ★★★★☆ (Élevée)

Récompenses : Équipement légendaire, œuf de dragon

Description : Une forteresse imprenable où réside un dragon légendaire. Seuls les plus braves osent s'y aventurer, car les ennemis sont nombreux et puissants.

Conseil : Formez une équipe équilibrée avec des soigneurs pour affronter les vagues d'ennemis avant le combat contre le dragon.

Caverne des Esprits

Type : Coopération

Niveau recommandé : 15-25

Difficulté : ★★★☆☆ (Modérée)

Récompenses : Artefacts anciens, potions rares

Description : Une caverne hantée par les esprits des aventuriers déchus. Les joueurs doivent affronter des ennemis fantomatiques tout en naviguant dans un environnement dangereux.

Conseil : Préparez des objets contre les effets de statut comme la paralysie ou la peur.

Ruines de l'Ancien Royaume

Type : Groupe

Niveau recommandé : 25-35

Difficulté : ★★★★☆ (Difficile)

Récompenses : Trésors royaux, artefacts magiques

Description : Plongez au cœur des ruines d'un royaume perdu, où chaque couloir est piégé et rempli d'énigmes mortelles.

Conseil : Amenez un voleur ou un éclaireur pour désactiver les pièges et repérer les embuscades.

Catacombes de l'Apocalypse

Type : Coopération

Niveau recommandé : 40+

Difficulté : ★★★★★ (Extrême)

Récompenses : Équipement mythique, reliques apocalyptiques

Description : Une descente aux enfers où des hordes de morts-vivants attendent de dévorer les intrus. Seuls les aventuriers les plus aguerris pourront survivre à ces profondeurs maudites.

Conseil : Formez une équipe avec des classes de soutien puissantes et prévoyez des stratégies défensives contre les vagues massives d'ennemis.
